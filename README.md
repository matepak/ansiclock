# ansiclock

Terminal clock writed in node inspired by tty-clock by Martin Duquesnoy <xorg62@gmail.com>.

# Installation

```
$ git clone https://github.com/matepak/ansiclock.git
$ npm install -g
```
# Usage
```
$ ansiclock
$ ansicock -c [red|green|yellow|blue|magenta|cyan|white]
```
